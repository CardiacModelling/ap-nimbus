appredict-with-emulators
============================

|appredict-with-emulators| is a container which has |appredict-no-emulators| as its
foundation, with all the Lookup Tables available at build time pre-installed to
avoid the need to dynamically download them. 
 * appredict-with-emulators is available in the ap-nimbus git repository via the appredict-docker submodule
