client-direct
=============

|client-direct| is a `Django 4 <https://www.djangoproject.com/>`_ component of |AP-Nimbus|.

 * The client is available at https://github.com/CardiacModelling/ap-nimbus-client.

 * Conventionally |client-direct| isn't installed/run as stand-alone as it generally serves no purpose to install
   only the |UI|. |br|
   For installing details see  :ref:`Install cardiacmodelling/ap-nimbus-client-direct <installation-client-direct>` |br|
   Developers may be interested in further reading in the developer section of
   :ref:`Developer client-direct section <developer-container-client-direct>`.
